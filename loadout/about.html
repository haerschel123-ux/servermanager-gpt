About

This tool builds player spawn loadouts for DayZ servers using the Spawning Gear Configuration (cfgPlayerSpawnGearPresets) introduced in DayZ 1.19+.

How to use it on your server
  1. Build your loadout in the Generator and Download JSON.
  2. Upload the file to your mission folder (e.g. mpmissions/dayzOffline.chernarusplus/).
  3. Register it in cfggameplay.json under WorldsData → playerSpawnGearPresetFiles, e.g.
    "playerSpawnGearPresetFiles": [ "my_loadout.json" ]
  4. Restart the server. New characters now spawn with the loadout (if several presets are listed, one is picked by spawnWeight).
Notes
  • The item database (1431 items) was generated from types.xml, cfgspawnabletypes.xml and cfgrandompresets.xml.
  • Everything runs in your browser – saved loadouts live in local storage on this device.
  • Multiple items in the same slot are alternatives: the game picks one at random, weighted by each item's spawn weight.
