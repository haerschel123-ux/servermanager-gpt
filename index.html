    ℹ️ Wichtig zu wissen

    • Änderungen gelten erst nach einem Server-Neustart. Ablauf: bearbeiten → 💾 Speichern → 🔄 Neustarten – dann ist alles im Spiel.
    • Die Tools ändern nur das, was du eingibst – der Rest der Datei bleibt unangetastet. Die 🔍 Vorschau zeigt dir vorher jede Änderung als Diff.
    • Vor jedem Speichern legt das Tool automatisch ein Backup an – am PC im Ordner backups/, am Handy im Browser (abrufbar unter ⚙️ → Backups).
    • Der Dateimanager zeigt alle Dateien, die wirklich auf deinem Server liegen – auch die, die das Nitrado-Webinterface nicht anzeigt.

    Fertige Generatoren für beliebte Server-Anpassungen. Ablauf: konfigurieren → 🔍 Vorschau (zeigt genau, welche Datei sich wie ändert) → Übernehmen → unten 💾 Speichern → Server neu starten, fertig.
