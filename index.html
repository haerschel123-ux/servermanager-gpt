    ℹ️ Wichtig zu wissen (Konsolen-Server)

    • Eine init.c gibt es auf PS4/PS5/Xbox-Servern nicht – eigenes Scripting ist nur auf PC-Servern möglich. Dieses Tool zeigt dir dafür alle Dateien, die wirklich auf deinem Server liegen (auch die, die das Nitrado-Webinterface nicht anzeigt).
    • Spieler live teleportieren oder Items sofort spawnen geht auf Konsole nicht – dafür bräuchte es Server-Mods, die nur auf PC laufen.
    • Was geht: Spawnpunkte, Events (z.B. Heli-Crashes), Gebäude/Objekte (Objekt-Spawner) und die komplette Loot-Tabelle ändern. Nach dem Speichern einmal neustarten – dann ist alles im Spiel.
    • Vor jedem Speichern legt das Tool automatisch ein Backup an – am PC im Ordner backups/, am Handy im Browser (abrufbar unter ⚙️ → Backups).
